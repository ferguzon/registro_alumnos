Se cambia información del botón y mensaje de activo o inactivo del formulario de mostrar carrera

Versión 0.1, 26 de abril de 2017

En esta versión, el inicio de sesión se controla de manera apropiada. Además, se impide
que el usuario se salte a otra página si no ha iniciado sesión.

El sistema incluye las opciones de agregar y actualizar usuarios, agregar y mostrar
carreras, generar un informe de carreras, volver al inicio y cerrar sesión.
